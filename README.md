# Autonomous UGV for Unstructured Environments

## Overview

This project aims to develop a fully autonomous UGV capable of navigating safely through unstructured environments using stereo vision, classical computer vision algorithms, and deep learning.

## Key Features

- Visual SLAM
- Terrain Classification
- Semantic Segmentation
- Obstacle Avoidance (WIP)
- Safety Redundancy Layer

## Hardware

- NVIDIA Jetson AGX Orin
- ZED X Stereo Camera
- Electric Motorcycle Platform
